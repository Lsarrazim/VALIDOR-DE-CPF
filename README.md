# Validador de CPF

Este projeto em python permite validar números de CPF.
Ele verifica:
- se o cpf digitado tem 11 números
- se todos os caracteres são numeros
- se o 10 e 11 numero são validos
  
# funcionalidades
- Verifica se o CPF tem 11 dígitos
- Valida apenas números
- Calcula o primeiro e segundo dígito verificadores
- Informa se o CPF é válido ou inválido
